Imagens Dinâmicas

Tente diminuir o tamanho do navegador ou abrir o site em outros dispositivos com a resolução menor.

Reproduzindo áudio

Vamos aprender as duas maneiras de reproduzir áudio em html5

Para reproduzir audio precisamos usar a tag audio, depois os parâmetros controls e autoplay

A segunda maneira é usando a tag audio mas adicionando sources src depois é só preencher normalmente e informar o tipo de arquivo do audio

Reproduzindo vídeos no site

Assim como nos áudios, existem duas maneiras para colocar vídeos no site, precisamos usar a tag video, assimilada com outros parâmetros como control, autoplay, etc

Também podemos usar a tag video, mas usando os sources, o que torna compatível para cada navegador

Também podemos incorporar um vídeo do youtube
