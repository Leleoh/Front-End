Listas Ordenadas

As listas ordenadas são aquelas onde a ordem dos itens é algo muito importante, nelas é necessária a classificação dos itens, ordem de importância, prioridades, rankeamento, etc.
Para criar uma lista ORDENADA nós usamos a tag ol e a tag li para identificar cada item da lista

A tag ol possui um parâmetro do type, onde podemos alterar a identificação dos itens, ABCDE, I II III IV, 12345 etc

  1. Baki
  2. Yujiro
  3. Musashi
  4. Pickle

Listas Não Ordenadas

As listas não ordenadas são aquelas onde a ordem dos itens não é tão importante, por exemplo uma lista de compras no mercado, tarefas a fazer sem horário definido, lembretes, etc.

Para marcar uma lista não ordenada precisamos usar a tag ul para definí-la e a tag li para criar os itens

  • Chocolate
  • Arroz
  • Feijão
  • Coca-Cola

Também é possível crirar listas mistas, onde uma vai dentro da outra

  1. Item 1
  2. Item 2
    • Item 2.1
    • Item 2.2
  3. Item 3
