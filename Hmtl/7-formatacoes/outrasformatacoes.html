Podemos usar também formatações para códigos de outras linguagem usando a tag code

num = int(('Digite a sua idade')) if num < 18: print(f'Você é menor de idade') else: print(f'Você é maior de idade')

Todavia, desse modo o código não fica formatado, para formatar em formato de código é necessário usar a tag pre juntamente da tag code


        num = int(('Digite a sua idade'))
        if num < 18:
            print(f'Você é menor de idade')
        else:
            print(f'Você é maior de idade')
