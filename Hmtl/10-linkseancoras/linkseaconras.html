Usando Links Externos

Podemos usar links em cima de textos com a tag a, e depois referenciando com uma URL

Você pode acessar meu repositório público no GitHub

Quando fazemos isso, o link nos leva para fora do nosso site, mas existe a marcação target, que permite abrir uma nova janela com o link sem sair do nosso site

Você também pode acessar o meu canal no youtube

Para isso, temos que adicionar o target com marcações "_blank"
